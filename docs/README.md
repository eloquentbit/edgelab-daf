# Documentation

This directory contains the architectural and technical documentation of the EdgeLab Data Acquisition Framework (DAF).

## Structure

| Directory | Description |
|-----------|-------------|
| `architecture/` | Framework architecture and design. |
| `api/` | Public API specification. |
| `guides/` | Integration and developer guides. |
| `adapters/` | Platform Adapter specifications. |
| `branding/` | Vision, mission and project manifesto. |
| `decisions/` | Architecture Decision Records (ADR). |

Documentation evolves together with the implementation and is considered an integral part of the project.
