# Tests

This directory contains the automated test suite for the EdgeLab Data Acquisition Framework (DAF).

Testing is considered a fundamental part of the project and evolves together with the implementation.

The test suite will progressively include:

- Unit tests for Core components.
- Integration tests for Platform Adapters.
- Dataset validation tests.
- Exporter verification.
- Regression tests.

The primary objective is to ensure that the framework consistently preserves trading facts and produces reliable canonical trade datasets across all supported trading platforms.
